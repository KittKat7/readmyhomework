# Read My Homework

**Awarded: AppalHack 2026 2nd place**

This project was created for Appal Hack 2026.  The main branch will continue to
recieve updates. There is a branch "AppHack2026Final" which will hold the final
version of code from the event. The documentation in that branch may be updated,
but the code will not, its purpose is to document the state of the project from
the event.

## Background

I learn better from listening than reading. Over my years in college, I've found
many ways to convert reading assignments and instructions into a listenable
format. This project aims to streamline the workflow I've been improving for
several years. This is my second attempt. My first was during my first year of
college, and my text-to-speech flow back then was not great. For the 2026 Appal
Hack event, knowing I'd be short on time (unable to attend the entirety of the
event), it seemed like a great opportunity to try again with my new
text-to-speech workflow. And? I think it turned out great!

## Features

- Easy to navigate interface with intuitive user flow.
- Perform optical character recognition on uploaded PDFs to extract text from
embeded images.
- Run text-to-speach on text extracted from the PDF.

### Planned Features
